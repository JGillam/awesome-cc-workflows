# Awesome Claude Code Workflows

A curated collection of Claude Code configurations, workflows, and resources to enhance your AI-assisted development experience.

## About

This repository contains practical Claude Code tools and workflows, organized by functional area. It includes both original content (slash commands, hooks, configurations) and curated links to third-party resources.

## How to Use

Each tool or workflow includes usage instructions and examples. For details on installing slash commands, hooks, or MCP servers, refer to the [official Claude Code documentation](https://code.claude.com/docs/en/sub-agents).

## Contents

### Session Management

Maintain context and continuity between Claude Code sessions.

- **[tidyup & tidydown](session-management/tidyup-tidydown.md)** - Slash command pair for end-of-session cleanup and start-of-session context loading. Automatically creates session summaries to seamlessly pick up where you left off.

## Contributing

This repository grows organically as new tools and workflows are developed. Contributions are welcome! Feel free to submit your own Claude Code workflows, configurations, or links to useful resources.
